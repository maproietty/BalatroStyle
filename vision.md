# BalatroStyle — Gameplay Vision & Design Notes

This file is yours to write in freely. No special formatting required.
Claude reads it at the start of every sprint so your intentions shape the
code — not just the checklist. The more you add here, the more the game
feels like *your* game.

Write like you're describing the game to a friend. What excites you about it?
What should a big scoring moment feel like? What makes you put the game down
satisfied vs frustrated?
